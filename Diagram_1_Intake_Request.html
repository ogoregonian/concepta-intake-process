AI Agent
📌 Prerequisites
  • Basecamp Request Board integration — AI Agent can read submissions and move items between columns
  • Zoho Sprints integration — AI Agent can generate tickets, assign to epics, and tag assignees
  • Backlog Sprint Buckets configured: Reported Issues · New Requests
  • Epics configured: Production Support · Planning/Ops/New Requests
  • Account Service budget buckets defined and baselined (Production Support · Planning/Ops/New Requests · Maintenance · Dev · Deployments/Releases)
  • SLA response definitions and severity level definitions loaded per account — used to determine notification content and routing
  • Slack channel configured with assignees mapped per request type
  • Notification templates configured per request type and routing outcome
🤖 AI Agent scope in this lane: Picks up all new submissions from the Request Board · Validates completeness of intake data · Loops back to requestor if data is missing · Categorises the request type from the data values · Generates the appropriate ticket in Zoho Sprints · Moves the Basecamp card to the correct column · Notifies the requestor and tags the assigned team in Slack · Hands off to the Internal Team for verification. All actions are rule-based — no autonomous decisions.
