AI Agent
📌 Prerequisites
  • Zoho Sprints: New Requests backlog sprint bucket configured · Planning/Ops/New Requests epic active with baselined budget hours
  • Basecamp Request Board: "Analyzing / Scoping / Estimating" and "Review Scope / Approve / Prioritize" columns active
  • Slack channel active with Analysis Team (PDM · SA) as primary assignees
  • Planning ceremony schedule loaded — used to determine expected activity windows and flag inactivity
  • SLA response definitions and notification templates configured per account
  • Budget threshold alert level defined — percentage of Planning/Ops budget that triggers a PDM flag
🤖 AI Agent scope in this lane: Handles the initial handoff to the Analysis Team, then runs a lightweight continuous monitoring role throughout the workflow — tracking ticket activity, nudging stalled steps, relaying requestor feedback, and flagging budget thresholds. All actions are rule-based triggers, not autonomous decisions. The goal is to remove coordination overhead from the PDM and SA so they stay focused on the analysis work itself.

Analysis Team
📌 Prerequisites
  • DoD (Definition of Done) defined and aligned with all stakeholders
  • Internal & external alignment on new operational structure for processing requests
  • Account budget structure established · Planning/Ops/New Requests bucket active
  • Planning ceremony schedule defined with committed resources
  • Smartsheet Project Plan template configured per account — initiated at the start of analysis to track scope, activities, timeline, resource allocation, and delivery milestones
  • Smartsheet Estimator template configured — must account for all activities required to achieve DoD (scoping, development, QA, UAT, deployment, documentation, etc.)
  • Zoho Sprints access to manage ticket lifecycle and log activities
  • Basecamp Request Board access to manage column movement and associate artifacts
  • Notion internal board (per account) — working surface for request analysis, requirements capture, T-shirt sizing, LOE estimates, and supporting artifacts
